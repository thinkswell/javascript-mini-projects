# 🎨 Color Changer

A simple interactive **Color Changer** built using **HTML, CSS and JavaScript**.  
Users can generate colors using RGB sliders, select from predefined palettes, enter custom color codes, enjoy smooth animations and download or share their color schemes.

---

### 🚀 Features

- **RGB Color Sliders**
  - Adjust Red (R), Green (G) and Blue (B) values (0–255)
  - Real-time background color updates
  - Displays current `rgb()` value

---

### ✏️ Enhancements & Changes Made

- **Custom Color Input**
- Users can input their own color codes:
  - HEX (`#ff5733`)
  - RGB (`rgb(255, 87, 51)`)
  - Named colors (`blue`, `teal`, etc.)
- Input validation to prevent invalid color values
- Error feedback shown for incorrect formats

- **Animated Color Transitions**
- Smooth fade animation when changing colors
- Animations work consistently across:
  - RGB sliders
  - Palette clicks
  - Custom color input

---

### 📤 Download & Share Color Schemes
- **Download color scheme**
  - Saves the selected color as a `.txt` file
  
- **Share color**
  - Share the selected color template

---

### 🛠️ Technologies Used
- **HTML5** – Structure
- **CSS3** – Styling and animations
- **JavaScript (Vanilla)** – Interactivity and logic
