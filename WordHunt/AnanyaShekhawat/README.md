Project: Word Hunt 

Overview: The game is a simple word game where the player select letters from the board and connects them to form words. The pbjective is to form as many words as possible before the timer runs outs. Complcated words (words 6+ letters) are given more points compared to simpler words  

Frameworks/Libraries: None 
APIs: DictionaryAPI.dev - fre dictrionary api with no APi key required 

Run Locally: 
1. Install `http-server` globally: 
     npm install -g http-server
2. cd project directory 
3. http-server -p 8000
4. Open http://localhost:8000 on local browser
